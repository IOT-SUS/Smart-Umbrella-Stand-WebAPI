## File Structure
```bash
config      # 所有設定資料 ex: secret token
controllers # 所有API的邏輯
models      # 資料庫相關
middlewares # 帳戶驗證相關
libs        # 所有Tools
static      # 所有public的東西 ex: css, js, images...
templates   # 所有html
router.py   # 所有路徑的定義
app.py      # 資料庫相關
run.py      # 程式的進入點
README.md
```
